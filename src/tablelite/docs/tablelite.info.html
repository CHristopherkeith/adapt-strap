### The column definition object is an array of definition objects ### The definition object should have properties shown in the example below: { columnHeaderDisplayName: 'String', // Not Required if columnHeaderTemplate is specified columnHeaderTemplate: 'String', // Not Required if columnHeaderDisplayName is specified. displayProperty: 'String', // Required if sortable is true. if templateUrl is missing, this property will be rendered in each cell cellFilter: 'String' // - Optional - the filter expression to apply to the displayProperty templateUrl: 'String' // - Not required if displayProperty is specified - look at artistPicture.html, sortable: 'Boolean' // - Optional - if true, cell becomes sortable }
