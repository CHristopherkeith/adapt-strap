### The column definition object is an array of definition objects ### The definition object should have properties shown in the example below: { columnHeaderDisplayName: 'String', // Not Required if columnHeaderTemplate is specified columnHeaderTemplate: 'String', // Not Required if columnHeaderDisplayName is specified. cellFilter: 'String' // - Optional - the filter expression to apply to the displayProperty templateUrl: 'String' // - Not required if displayProperty is specified - look at artistPicture.html }
